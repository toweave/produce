# Seedance-1.5 视频创作功能 — 产品需求文档

## 1. 概述

- **背景**：在桌面应用中集成豆包 Seedance-1.5-Pro 模型的视频生成能力，让用户可以直接在应用内完成 AI 视频创作的全流程，无需切换到浏览器或使用命令行。
- **目标用户**：使用本工具进行 AI 视频创作的用户，频率为中高频（日均多次使用）
- **成功指标**：用户能独立完成"创建视频任务 → 跟踪进度 → 查看/下载结果 → 管理历史任务"的完整闭环

## 2. 用户故事

### P0（第 1 迭代）

**用户故事 1：视频创作**
> 作为创作者，我希望在侧边栏看到 Seedance-1.5 菜单组，以便快速进入视频创作功能

**验收标准**：
- Given 我在应用主界面，When 我展开侧边栏，Then 可以看到 "Seedance-1.5" 菜单组，展开后包含"视频创作"和"任务列表"两个子菜单

**用户故事 2：创建视频生成任务**
> 作为创作者，我希望通过文本提示词生成视频，以便将创意快速转化为视频内容

**验收标准**：
- Given 我进入"视频创作"页面，When 我输入提示词并点击"生成"，Then 系统创建视频生成任务并返回任务 ID
- Given 我进入"视频创作"页面，When 我未输入提示词就点击"生成"，Then 系统提示"请输入视频提示词"
- Given 我正在创建任务，When 系统正在提交请求，Then 按钮显示加载状态且不可重复点击

**用户故事 3：查看任务状态**
> 作为创作者，我希望查看提交的任务状态，以便了解视频是否生成完成

**验收标准**：
- Given 我进入"任务详情"页面，When 任务正在排队或运行中，Then 页面自动轮询更新状态
- Given 任务状态为 succeeded，When 我查看详情，Then 显示视频播放器，可直接播放和下载
- Given 任务状态为 failed，When 我查看详情，Then 显示错误信息和错误码

**用户故事 4：管理任务**
> 作为创作者，我希望查看所有任务列表并管理它们，以便跟踪我的创作历史

**验收标准**：
- Given 我进入"任务列表"页面，When 页面加载完成，Then 展示所有已创建的任务（ID、状态、创建时间、模型）
- Given 任务列表中，When 我点击某个任务的"查看"，Then 跳转到该任务详情页
- Given 任务状态为 queued/succeeded/failed/expired，When 我点击"删除"，Then 任务被取消或删除

### P1（第 2 迭代）

**用户故事 5：图生视频**
> 作为创作者，我希望上传参考图片作为视频首帧（或首尾帧），以便生成更符合预期的视频内容

**验收标准**：
- Given 我在创作页面，When 我选择本地图片作为首帧参考，Then 图片上传并显示预览
- Given 我勾选"首尾帧"模式，When 我选择两张图片分别作为首帧和尾帧，Then 系统按首尾帧模式创建任务

**用户故事 6：参数配置**
> 作为创作者，我希望调整视频参数（分辨率、宽高比、时长等），以便控制输出规格

**验收标准**：
- Given 我在创作页面，When 我选择 1080p 分辨率，Then 生成 1080p 视频
- Given 我在创作页面，When 我选择"自适应"宽高比，Then 模型根据输入自动选择

### P2（后续迭代）

- 任务完成时发送系统桌面通知
- 支持样片模式（draft mode），先用低成本预览再生成正式视频
- 支持将生成的视频保存到本地指定路径
- 任务列表支持搜索和高级筛选

## 3. 功能流程

### 主路径：视频创作

```
侧边栏 "Seedance-1.5" → "视频创作"
    ↓
填写提示词（必填）
    ↓
可选：上传参考图片 / 配置参数（分辨率、宽高比、时长、音频、水印）
    ↓
点击"生成视频"
    ↓
系统验证输入 → 调用 API 创建任务
    ↓
创建成功 → 显示任务 ID → 提供"查看详情"和"继续创作"两个操作
    ↓
查看详情 → 跳转到任务详情页 → 自动轮询 → 播放视频
```

### 主路径：任务管理

```
侧边栏 "Seedance-1.5" → "任务列表"
    ↓
展示所有任务（ID、状态、创建时间、模型）
    ↓
按状态筛选（全部/排队中/运行中/已完成/失败/已取消/已过期）
    ↓
点击任务 → 跳转任务详情
    ↓
查看视频 / 参数信息 / 删除任务
```

### 异常路径

| 场景 | 处理方式 |
|------|---------|
| **网络断开** | 提示"网络连接失败，请检查网络后重试" |
| **API Key 未配置** | 提示"请先配置 SEE_DANCE_API_KEY 环境变量" |
| **创建任务失败** | 显示错误信息（如余额不足、模型未开通、参数错误） |
| **查询任务失败** | 显示错误提示，提供"刷新"按钮 |
| **任务长时间无响应** | 用户可手动刷新页面；任务超时后状态变为 expired |
| **视频 URL 过期** | 提示"视频链接已过期（有效期 24 小时）"，建议及时下载 |

### 首次使用

- 首次进入"视频创作"页面时，提示词输入框为空，显示 placeholder 示例文本
- 首次进入"任务列表"页面时，显示空状态提示"暂无任务"

## 4. 需要的桌面能力

| 能力 | 用途 |
|------|------|
| **文件对话框** | 选择本地图片作为参考帧（首帧/首尾帧） |
| **拖拽交互** | （P2）拖拽图片到创作页面上传 |
| **系统通知** | （P2）任务完成时发送桌面通知 |

## 5. 依赖项

| 依赖 | 说明 |
|------|------|
| **火山引擎方舟平台 API** | 调用视频生成服务。需要先注册并获取 API Key（https://console.volcengine.com/ark/region:ark+cn-beijing/apikey） |
| **模型开通** | 需要在方舟平台开通 `doubao-seedance-1-5-pro-251215` 模型服务 |
| **账户余额** | 确保方舟平台账户余额大于等于 200 元，或已购买资源包 |
| **环境变量** | `SEE_DANCE_API_KEY` 需要在运行应用时注入 |

**文档参考**：方舟平台提供了完整的 [视频生成 API 文档](https://www.volcengine.com/docs/82379/1520757) 和 [调用教程](https://www.volcengine.com/docs/82379/1366799)，包含 Curl/Python/Java/Go 示例。

## 6. 非功能需求

- **响应时间**：创建任务应在 3 秒内返回任务 ID；视频生成本身为异步过程（时长取决于模型负载和规格，通常在数十秒到数分钟）
- **轮询策略**：任务排队/运行中时，每 5 秒轮询一次状态
- **兼容性**：Windows / macOS / Linux 三端行为一致
- **可测试性**：所有 API 调用可通过 IPC 隔离测试，UI 可使用 mock 数据验证
