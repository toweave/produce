# Seedance-1.5 视频创作页面 UI 优化 — 产品需求文档

## 1. 概述

- **背景**：当前关键帧区域的"手动截图"标题和"清空全部"按钮分属不同位置，首尾帧开关默认关闭，引用关键帧只能设为首帧。需要优化这三处交互细节。
- **目标用户**：所有使用视频创作页面的创作者。
- **成功指标**：
  - "手动截图"标签与"清空全部"按钮在同一行右对齐，布局更紧凑
  - "使用首尾帧"默认开启，减少用户每次手动勾选的操作
  - 引用关键帧时可选择设为首帧或尾帧，满足首尾帧参考需求

## 2. 用户故事

### P0

**用户故事 1：手动截图标题与清空全部同行**
> 作为创作者，我希望"手动截图"标题与"清空全部"按钮在同一行，以便界面布局更紧凑。

**验收标准**：
- Given 关键帧区域有手动截图和清空全部按钮，When 我查看页面，Then "手动截图"标签在左、"清空全部"按钮在同一行右侧
- Given 只有自动关键帧没有手动截图，When 我查看页面，Then "清空全部"按钮在自动关键帧区域上方右对齐

**用户故事 2：使用首尾帧默认为开启**
> 作为创作者，我希望"使用首尾帧"选项默认为勾选状态，以便减少每次都要手动开启的操作。

**验收标准**：
- Given 我首次进入创作页面，When 页面加载完成，Then "使用首尾帧"复选框默认为勾选状态
- Given 我勾选了首尾帧并上传了尾帧图片，When 表单参数自动保存后再次进入，Then 恢复勾选状态和尾帧图片

**用户故事 3：引用图片时可选引用为首帧或尾帧**
> 作为创作者，我希望在关键帧上悬停时可以选择引用为"首帧"或"尾帧"，以便灵活设置参考图。

**验收标准**：
- Given 我鼠标悬停在关键帧缩略图上，When 查看操作菜单，Then 显示"引用为首帧"和"引用为尾帧"两个按钮
- Given 我点击"引用为首帧"，When 操作完成，Then 该关键帧图片替换左栏的首帧参考图
- Given 我点击"引用为尾帧"，When 操作完成，Then 该关键帧图片替换左栏的尾帧参考图，且"使用首尾帧"自动勾选

## 3. 功能流程

### 主路径：引用关键帧为首帧/尾帧

```
鼠标悬停在关键帧缩略图上
    ↓
显示"引用为首帧""引用为尾帧"按钮
    ↓
点击"引用为首帧" → 替换左栏首帧图（imageData）
点击"引用为尾帧" → 替换左栏尾帧图（lastFrameData），自动勾选"使用首尾帧"
    ↓
左栏参考图区域更新
```

### 异常路径

| 场景 | 处理方式 |
|------|---------|
| **引用为尾帧时首帧图未上传** | 尾帧图照常设置，首帧图保持为空，由用户自行上传或引用 |
| **手动截图区域为空，只有自动关键帧** | "清空全部"按钮在自动关键帧区域的标题行右侧 |
