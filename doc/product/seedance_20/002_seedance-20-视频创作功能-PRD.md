# Seedance 2.0 视频创作功能 — 产品需求文档

## 1. 概述

- **背景与动机**：火山引擎方舟平台发布了 Doubao Seedance 2.0 系列模型，相比 1.5 版本新增了多模态参考（图片/视频/音频组合输入）、编辑视频、延长视频、联网搜索等能力，视频时长延长至 15 秒，支持自适应宽高比。需要在桌面端集成新模型能力，与现有的 Seedance 1.5 菜单组并行存在。
- **目标用户**：已有视频创作需求的桌面端用户，需要升级到 Seedance 2.0 模型以获得更好的生成质量和更多模态输入。
- **成功指标**：用户能顺畅地使用 Seedance 2.0 模型完成视频创作全流程（创建任务 → 查看进度 → 查看结果 → 管理任务）。

## 2. 用户故事

### P0

#### US1：创建视频生成任务
- **作为** 用户，**我希望** 使用 Doubao Seedance 2.0 模型创建视频生成任务，**以便** 生成高质量视频。
- **验收标准**：
  - Given 我已正确配置 API Key，当我填写文本提示词并配置参数后点击"生成"，系统应调用 Seedance 2.0 API 创建任务并返回任务 ID
  - Given 我选择了"多模态参考"模式，当我上传参考图片/视频/音频后点击"生成"，系统应将多模态素材作为 content 数组传入 API
  - Given 我开启"联网搜索"开关，当 API 调用时应在 tools 参数中包含 `web_search`
  - Given 我未配置 API Key 或密钥无效，当我点击"生成"时系统应给出明确错误提示
  - Given API 返回错误，系统应展示错误信息

#### US2：查询视频生成任务
- **作为** 用户，**我希望** 查看某个视频生成任务的详情，**以便** 了解生成进度或查看生成的视频。
- **验收标准**：
  - Given 任务已完成（succeeded），页面应展示生成的视频播放器
  - Given 任务失败（failed），页面应展示错误信息
  - Given 任务排队中/运行中，页面应提示当前状态并建议稍后刷新
  - Given 任务已取消或过期，页面应展示对应状态

#### US3：查询视频生成任务列表
- **作为** 用户，**我希望** 查看我创建的所有视频生成任务，**以便** 管理和追踪任务。
- **验收标准**：
  - Given 我有多个任务，列表应分页展示，每页 20 条
  - Given 我选择某个状态筛选条件，列表应只展示对应状态的任务
  - Given 没有任务时，列表应显示空状态提示

#### US4：取消或删除视频生成任务
- **作为** 用户，**我希望** 取消排队中的任务或删除已完成/失败/过期的任务记录，**以便** 清理任务列表。
- **验收标准**：
  - Given 任务处于 queued 状态，调用 DELETE 后任务变为 cancelled
  - Given 任务处于 succeeded/failed/expired 状态，调用 DELETE 后删除任务记录
  - Given 任务处于 running/cancelled 状态，不应显示删除按钮（DELETE 不支持）

### P1

#### US5：多模态参考输入
- **作为** 用户，**我希望** 上传参考图片（0-9 张）、参考视频（0-3 个）、参考音频（0-3 个）作为输入素材，**以便** 控制生成视频的内容风格。
- **验收标准**：
  - Given 我上传了参考图片，content 数组中应包含 `type: "image_url"` 且 `role: "reference_image"`
  - Given 我上传了参考视频，content 数组中应包含 `type: "video_url"` 且 `role: "reference_video"`
  - Given 我上传了参考音频，content 数组中应包含 `type: "audio_url"` 且 `role: "reference_audio"`
  - Given 我只上传了音频（无图片/视频），应提示用户至少需要一张图片或一个视频

#### US6：联网搜索
- **作为** 用户，**我希望** 开启联网搜索功能，**以便** 模型能搜索最新互联网信息辅助生成视频。
- **验收标准**：
  - Given 我开启了联网搜索开关，API 请求中应包含 `tools: [{type: "web_search"}]`
  - Given 联网搜索仅支持纯文本输入模式

### P2

（后续迭代）
- 视频编辑功能（替换主体、增删对象）
- 视频延长功能（向前/向后延长、多段拼接）
- 首尾帧图生视频
- 虚拟人像素材选择
- priority 优先级设置

## 3. 功能流程

### 主路径：视频创作

1. 用户点击侧边栏 "Seedance-2.0" → "视频创作" 进入创建页面
2. 用户选择模型（Seedance 2.0 / Seedance 2.0 Fast）
3. 用户输入文本提示词
4. 用户可选上传参考图片、参考视频、参考音频
5. 用户配置参数：宽高比（含 adaptive）、时长（4-15s）、分辨率、生成音频开关、水印开关、联网搜索开关、返回尾帧开关
6. 用户点击"生成"按钮
7. 系统调用 API 创建任务 → 显示加载状态 → 成功后展示任务详情链接
8. 用户可在任务列表页查看所有任务，在详情页查看视频结果

### 主路径：任务管理

1. 用户点击侧边栏 "Seedance-2.0" → "任务列表" 进入列表页
2. 系统分页展示任务列表（ID、状态、模型、创建时间）
3. 用户可按状态筛选（全部/排队中/运行中/已完成/失败/已取消/已过期）
4. 用户点击"查看"进入详情页查看视频
5. 用户点击"删除"删除符合条件的任务记录

### 异常路径

- **网络断连**：API 调用超时或网络错误，展示错误提示，用户可重试
- **API Key 未配置**：弹窗或错误提示，引导用户配置环境变量
- **素材文件过大**：Base64 编码时请求体超过 64MB，提示用户使用公网 URL
- **任务过期**：任务超过 48 小时未完成自动过期，详情页展示 expired 状态
- **视频 URL 过期**：视频 URL 有效期 24 小时，过期后无法播放，提示用户重新查询

## 4. 需要的桌面能力

| 桌面能力 | 说明 |
|---------|------|
| **文件对话框** | 选择本地图片/视频/音频文件上传作为参考素材 |
| **文件系统读写** | 读取本地文件并转为 Base64 编码或文件路径 |
| **系统通知** | （可选）任务完成时发送桌面通知 |

## 5. 依赖项

- **外部 API**：火山引擎方舟平台 Seedance 2.0 API
  - 创建任务：`POST /api/v3/contents/generations/tasks`
  - 查询任务：`GET /api/v3/contents/generations/tasks/{id}`
  - 任务列表：`GET /api/v3/contents/generations/tasks?page_num=&page_size=&filter.status=`
  - 删除任务：`DELETE /api/v3/contents/generations/tasks/{id}`
- **API Base URL**：`https://ark.cn-beijing.volces.com/api/v3`
- **API Key**：与 Seedance 1.5 共用 `VITE_SEE_DANCE_API_KEY` 环境变量
- **模型 ID**：
  - `doubao-seedance-2-0-260128`（标准版）
  - `doubao-seedance-2-0-fast-260128`（快速版，不支持 1080p）
- **账户要求**：火山引擎账户余额 ≥ 200 元，或已购买资源包
- **CSP 要求**：需允许 `media-src https:` 播放外部视频
- **约束条件**：与 Seedance 1.5 并行存在，各自独立的菜单项和路由

## 6. 与 Seedance 1.5 的关键差异

| 维度 | Seedance 1.5 | Seedance 2.0 |
|------|-------------|-------------|
| 模型 ID | `doubao-seedance-1-5-pro-251215` | `doubao-seedance-2-0-260128` / `-fast-260128` |
| 输入模态 | 文本 + 图片（首帧/首尾帧） | 文本 + 图片(0-9) + 视频(0-3) + 音频(0-3) |
| 内容结构 | `image_url` 无 role 字段 | 需指定 `role: reference_image/video/audio` |
| 视频输入 | 不支持 | 支持（0-3 个参考视频） |
| 音频输入 | 不支持 | 支持（0-3 个参考音频） |
| 时长范围 | 4-12 秒 | 4-15 秒 |
| 宽高比默认值 | `16:9` | `adaptive` |
| 分辨率 | 480p/720p/1080p | 480p/720p/1080p（Fast 不支持 1080p） |
| 联网搜索 | 不支持 | 支持（纯文本模式） |
| priority | 不支持 | 支持（0-9） |
| return_last_frame | 不支持 | 支持 |
| camera_fixed | 不支持 | 不支持 |
| draft/样片 | 支持 | 不支持 |
| 编辑视频 | 不支持 | 支持（P1） |
| 延长视频 | 不支持 | 支持（P1） |

## 7. 非功能需求

- **性能**：列表页分页查询响应应在 3 秒内
- **兼容性**：与 Seedance 1.5 并行运行，不产生冲突
- **可用性**：表单验证在提交前给出明确错误提示；多模态素材上传有进度和预览
- **安全**：API Key 仅在主进程使用，不暴露给渲染进程；用户文件不上传到第三方服务器
